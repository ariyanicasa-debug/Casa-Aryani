# Casa Ariyani Website

This is the fixed flat GitHub Pages version.

Upload all files directly to your GitHub repository root.

Do not put the files inside another folder.

Required root files:
- index.html
- styles.css
- script.js
- all .jpg images
- casa-ariyani-logo.svg
- .nojekyll

GitHub Pages setup:
Settings > Pages > Deploy from branch > main > /root > Save

Booking button opens the official Google Form.


## Availability Sync
This version reads booked dates from your Google Sheet. If a row is deleted from the sheet, the date becomes available again after website refresh. See `AVAILABILITY-SYNC-GUIDE.txt`.
